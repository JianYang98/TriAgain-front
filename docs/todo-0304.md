앱스토어 출시 필수 요건.

### 변경 파일
- `pubspec.yaml` — `sign_in_with_apple: ^6.1.0` 추가
- `lib/providers/auth_provider.dart` — Apple 임시 프로바이더 3개 추가 (`appleIdentityTokenProvider`, `appleUserIdProvider`, `appleProfileProvider`)
- `lib/services/auth_service.dart` — `loginWithApple()` (POST /auth/apple) + `signupWithApple()` (POST /auth/signup/apple) 추가
- `lib/features/auth/screens/login_screen.dart` — Apple 로그인 버튼 (iOS만) + `_loginWithApple()` 플로우
- `lib/features/auth/screens/onboarding_screen.dart` — 카카오/Apple 겸용 (initState 닉네임 기본값 + `_signup()` 분기)

### Apple 로그인 플로우
```
Apple SDK → identityToken 획득
  → POST /auth/apple
    → 기존 유저 (isNewUser=false): 토큰 저장 → /home
    → 신규 유저 (isNewUser=true): 임시 저장 → /onboarding → POST /auth/signup/apple → /home
```
